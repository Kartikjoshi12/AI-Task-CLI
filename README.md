# AI Task CLI

AI-first task management from the terminal.

## Usage

```bash
npx task
```

## Development

```bash
npm run dev
```

## Scripts

| Command           | Description              |
| ----------------- | ------------------------ |
| `npm run build`   | Compile TypeScript       |
| `npm run dev`     | Run CLI in development   |
| `npm run lint`    | Lint source code         |
| `npm run format`  | Format source code       |
| `npm test`        | Run tests                |
